## 👨‍💻 Execução

```bash
## 1. Clone o projeto
git clone https://github.com/elanonc/books-crud

## 2. Entre no diretório do projeto
cd books-crud

## 3. Instale as dependências
npm install

## 4. Inicie o servidor
npm run start:dev

## 5. Execute as migrations
npx prisma migrate dev
```
